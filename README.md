## Run Locally

Clone the project

```bash
  git clone https://github.com/K1337xz/Movie-database-react.git
```

Go to the project directory

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

The server will start at: http://localhost:3000/
